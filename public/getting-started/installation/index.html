Installation

Requirements
- Vagrant has to be installed on your local machine.
- You need a valid SSH public key in ~/.ssh/id_rsa.pub to provision the clusters.

On Linux

git clone http://github.com/adaltas/jumbo jumbo
cd jumbo
pip install .
